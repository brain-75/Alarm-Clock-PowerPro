Alarm Clock PowerPro - Professional Portable Alarm / Timer / Scheduling Software

Version: 2026.07.07



SYSTEM REQUIREMENTS

	Windows 10 19045.x or later (64-bit)

	No installation required - fully portable

	It will run on older Win 10 but no support will be given to versions lower than 19045.x.



QUICK START

	Run "Alarm Clock Launcher.exe" to start.



INSTALLATION, UPDATES & DOWNGRADES 

	Optional backup:

		Back up /Data/Alarms/Alarms.xml and /Data/Config/Config.xml before updating.

	To update or downgrade:

		Exit the program, then extract the ZIP over the existing folder and replace existing files when prompted.



	Official ZIP release packages include only the Default*.xml files, not the live

	Alarms.xml or Config.xml user data files. Updating by extracting over an

	existing installation preserves your personal alarms and configuration.



	Failure to exit the program before updating may leave one or more EXE or DLL

	files partially updated, preventing the program from starting. Always exit the

	program before replacing files.



CLEAN INSTALL - PRESERVE DATA

	A clean install removes all program files and restores the default package

	while preserving your personal alarms and configuration.



	1. Exit the program.

	2. Back up:

			/Data/Alarms/Alarms.xml

			/Data/Config/Config.xml

			/Data/State/State.xml + State_old.xml			(optional)

	3. Delete the existing Alarm Clock PowerPro folder.

	4. Extract a fresh copy of the official full (60MB+) ZIP package (not the small 2-3MB patches).

	5. Restore your backed-up XML files to their original locations:

			/Data/Alarms/Alarms.xml

			/Data/Config/Config.xml

			/Data/State/State.xml + State_old.xml			(optional)



	Your alarms and settings will be restored while all program files are

	replaced with a clean copy. State files are optional and if omitted will be slowly regenerated.
